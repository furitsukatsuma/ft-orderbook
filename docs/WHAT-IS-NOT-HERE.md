# このリポジトリに含まれないもの

public ミラーは **使える・直せる・PR できる** 範囲だけを配布しています。  
メンテナーの private 運用や個人情報は **意図的に含めていません**。

## 含まれないもの（例）

| 項目 | 理由 |
|------|------|
| `docs/ops/`（keep-alive、Slack 月次、clone メモ） | 個人・組織の運用 |
| GitHub Actions Secrets の実値 | セキュリティ |
| Slack / Cursor ブリッジ | 個人インフラ |
| Cloudflare 本番デプロイ手順（メンテナーアカウント） | 個人運用 |
| 本番 Supabase プロジェクトへの接続情報 | BYOK — 各自で用意 |
| モノレポ全体（`ft-automation` private） | 製品以外の実験・自動化 |

## 配布方針

- **無料**（MIT）
- **利益はメンテナーが狙わない** — 各自ホスト・各自キー
- **友好的な開発者向け** — fork / PR 歓迎。write 招待は原則なし
- **見せる情報を最小化** — 使うのに必要なソースとドキュメントのみ

## 同期

正本の開発は private の `ft-automation` で行い、公開物はスクリプトでミラーします（メンテナー側）。
